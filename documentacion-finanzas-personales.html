Herramientas utilizadas: Gemini, Supabase y PWABuilder

Prompts:

1.Le indicamos una serie de coordenadas mapping que queremos

2.Le indicamos que nos de ideas para 5 secciones de nuestra app

3.Le indicamos que añada detalles visuales y técnicos en las diferentes secciones

4.Le indicamos cuando se deben mostrar los bloques de texto y que deben mostrar en diferentes secciones y otros detalles

5.Le indicamos que adapte el codigo para moviles pwa

6.Le indicamos las diferentes coordenadas de mapping segun si es movil o pc

7.Conexión con supabase para log-in y base de datos

9.Creamos el manifest

 

Supabase:

1.Tablas

2.Bucket para fotos

3.Autenticación

 

PWA:

2.Descargamos la versión android, por defecto(sin cambiar nada)

3.Finalmente el archivo apk lo deberemos instalar en u idspositivo android para asegurarnos que funciona
